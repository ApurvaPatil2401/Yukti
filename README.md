# Yukti – The Guarded IDE 
*A Pedagogy-First AI Coding Mentor*

## 🚀 Overview
**Yukti – The Guarded IDE** is an AI-powered coding mentor designed to prioritize **conceptual understanding over instant solutions**.  
Unlike traditional AI coding tools that immediately generate code, Yukti enforces a **learning-first approach** by requiring users to demonstrate understanding of core programming concepts before unlocking solutions.

> **Learning before output, not output before learning.**

##  Problem Statement
Modern AI coding tools (e.g., ChatGPT, GitHub Copilot) optimize for speed and convenience. While helpful, they often lead to:

- Over-dependence on AI-generated code  
- Superficial learning and copy-paste habits  
- Poor debugging skills  
- Weak performance in interviews and whiteboard problem solving  

This creates **“vibe coders”** — developers who can produce code but struggle to explain or reason about it.


## Solution
Yukti acts as a **strict AI mentor** that:

- Identifies the **core concepts** behind a coding problem  
- Asks **mandatory concept-based questions**  
- Evaluates user explanations in natural language  
- Provides **graduated hints** instead of instant answers  
- Unlocks the final solution **only after understanding is demonstrated**  

This approach helps learners build strong foundations and genuine problem-solving skills.


##  Key Features
-  **Solution Gate** – Prevents access to code until concept validation is complete  
-  **Concept Checker** – Evaluates explanations for accuracy, depth, and clarity  
-  **Adaptive Hint System** – Provides step-by-step guidance without revealing answers  
-  **Learning Tracker** – Tracks weak concepts and adapts future difficulty  
-  **Interview & Debugging Focus** – Designed for real-world problem solving, not shortcuts  


## How It Works
1. User submits a coding problem or buggy code  
2. System identifies required programming concepts  
3. User must explain concepts in natural language  
4. AI evaluates understanding  
   - If weak → hints & sub-questions  
   - If sufficient → solution unlocked  
5. Learning progress is stored and used for future sessions  

## 🛠️ Technology Stack (Proposed)
- **Programming Language:** Python  
- **AI / NLP:** Large Language Models (via AWS Bedrock)  
- **Backend:** Cloud-based services  
- **Data Storage:** User learning progress & concept history  


## Target Users
- Students learning Data Structures & Algorithms  
- Developers preparing for technical interviews  
- Learners who want **understanding, not shortcuts**  


## Impact
- Reduces unhealthy AI dependency  
- Improves conceptual clarity and debugging skills  
- Encourages active thinking and explanation  
- Supports learning-first AI adoption, especially for learners in Bharat
